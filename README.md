# Machine Translation for low-resource languages
NN: to run the scripts:
```shell
python3 -m venv mtenv
git clone https://github.com/anya-bel/low_resource_mt.git
cd low_resource_mt
pip3 install -r requirements.txt
cd scripts
bash hyper_params.sh
```
